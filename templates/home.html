What’s Inside

Key Features

📈

OLS Regression Model

Baseline predictive model trained on historical closing prices to estimate the next trading session’s target value.

⚡

Live Inference Engine

Real-time prediction using a trained machine learning model deployed through Flask for immediate T+1 estimation.

📊

Model Evaluation

Performance assessed using R² and MAE with a time-based split to support realistic out-of-sample evaluation.

🔬

Univariate Forecasting

Uses closing price as a single input variable to create a clean, interpretable baseline forecasting system.

🛡️

Input Validation

Prevents unrealistic values from reaching the model to preserve stable and reliable prediction behavior.

🖥️

Terminal Interface

Finance-inspired interface designed to simulate a practical analytical environment rather than a basic demo page.

The Project

Built for MSc Statistics with Data Science

This project applies supervised machine learning to financial time-series forecasting while keeping the system interpretable and usable.

The model was developed in Python using scikit-learn, deployed through Flask, and supported by a dashboard-oriented presentation layer to make the output easier to understand and explore.
